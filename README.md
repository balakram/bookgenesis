### Run Locally

To run **Bookgenesis** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/Bookgenesis.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/Bookgenesis.git
```
